# Frameworks on Vercel

Vercel supports all major frontend frameworks with zero configuration.

## Supported Frameworks

- **Next.js**: React framework with SSR, SSG, and API routes
- **Nuxt**: Vue framework with SSR and static generation
- **SvelteKit**: Svelte framework for building web apps
- **Astro**: Content-focused static site builder
- **Remix**: Full-stack React framework
- **Gatsby**: React-based static site generator
- **Vue.js**: Progressive JavaScript framework
- **Angular**: Platform for building mobile and desktop web apps

## Features

- Automatic framework detection
- Optimized build settings
- Built-in CI/CD
- Edge network deployment
- Serverless function support

## Quickstart

Deploy a new project with your framework:
```bash
npx create-next-app@latest my-app
```
