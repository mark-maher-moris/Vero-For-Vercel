# Integrations

Vercel Integrations extend your project with third-party services and tools.

## Integration Types

- **Analytics**: Web Analytics, Speed Insights
- **CMS**: Contentful, Sanity, Strapi
- **Commerce**: Shopify, BigCommerce
- **Databases**: Upstash, Neon, PlanetScale
- **Monitoring**: Sentry, Datadog
- **Communication**: Slack, Discord

## Marketplace

Browse and install integrations from the [Vercel Marketplace](https://vercel.com/marketplace).

## Creating Integrations

Build your own integration using the [Integration API](https://vercel.com/docs/integrations).

## Popular Integrations

- **MongoDB Atlas**: Cloud database
- **Cloudflare**: DNS and security
- **SendGrid**: Email delivery
- **Twilio**: SMS and communication
