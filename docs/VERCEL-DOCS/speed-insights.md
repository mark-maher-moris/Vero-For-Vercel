# Speed Insights Overview

Speed Insights is available on [all plans](https://vercel.com/docs/plans)

- To set up Speed Insights for your project, see the [Quickstart](https://vercel.com/docs/speed-insights/quickstart).
- To understand who is visiting your site, use [Web Analytics](https://vercel.com/docs/analytics).

Vercel Speed Insights provides you with a detailed view of your website's performance metrics, based on Core Web Vitals, enabling you to make data-driven decisions for optimizing your site.

## Features

- Core Web Vitals tracking (LCP, FID, CLS, TTFB, FCP)
- Real User Monitoring (RUM)
- Performance scoring
- Historical data trends
- Environment-specific tracking (preview and production)

## Dashboard View

The Speed Insights dashboard offers in-depth information about scores and individual metrics without the need for code modifications or leaving the Vercel dashboard.

## Getting Started

1. Enable Speed Insights in your project settings
2. Deploy your application
3. View metrics in the Vercel dashboard

For more details, see the [Quickstart guide](https://vercel.com/docs/speed-insights/quickstart).
